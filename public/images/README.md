# Images du Portfolio

## Images à ajouter

Remplacez les images suivantes par vos propres images :

### Hero Section
- **hero-bg.jpg** (1920x1080px) - Image de fond pour la section Hero

### Présentation
- **profile.jpg** (800x1000px) - Votre photo de profil

### Projets
- **project1.jpg** (800x600px) - Image du projet 1
- **project2.jpg** (800x600px) - Image du projet 2
- **project3.jpg** (800x600px) - Image du projet 3
- **project4.jpg** (800x600px) - Image du projet 4

### Galerie
- **gallery1.jpg** à **gallery8.jpg** (600x600px) - Images pour la galerie

### Témoignages
- **client1.jpg** à **client4.jpg** (200x200px) - Photos des clients

## Sources d'images gratuites

Vous pouvez utiliser des images de ces sites :
- [Unsplash](https://unsplash.com)
- [Pexels](https://pexels.com)
- [Pixabay](https://pixabay.com)

## Formats recommandés

- Format : JPG ou PNG
- Optimisez vos images avant de les ajouter (utilisez TinyPNG ou Squoosh)
- Utilisez des images de haute qualité pour un rendu professionnel
